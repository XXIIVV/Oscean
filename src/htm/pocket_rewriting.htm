A zine on multiset rewriting.

A eight-pager that introduces the concept of rewrite programming with a few examples. It also includes the source code for a tiny Fractran interpreter. The back cover is purposely left blank to allow for note-taking.

Playground

Try the examples above in this little playground, press the squiggly symbol transmute the symbols in the right-most panel, or press the arrow to reverse time and apply the rules backward.

You can also download the playground to use it locally with a Varvara emulator, or build it from the sources.
