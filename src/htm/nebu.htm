A spreadsheet editor.

Nebu is a graphical spreadsheet editor for the Varvara system, designed to read and write csv files. Math operations between cells are done by specifying a range of cells.

A range uses the colon character between two cell values, operations are made by specifying a range, followed by one of 4 arithmetic operator(+, -, *, /). For example: A5:C7*. There are also 2 extra operators(#, ") to get the length of a rectangle, and string concatenation. If an operator is not specified, it will default to returning the sum.
