Collected doodles made in a 4000 bytes program.

Oekaki is a somewhat incapable 2-bit drawing program for Varvara inspired from Noodle, with very even fewer features. The default chr image size is 40x28, or 512x320. It can open tga images. It was originally created to do close-ups for scenes of the Neauismetica.

Controls

  • ctrl+n New
  • ctrl+s Save
  • ctrl+o Open
  • tab Toggle menu
  • [] Brush size
  • 1234 Select color
  • qwer Select shape
  • asdf Select tool
  • h Flip horizontal

Custom Patterns & Brushes

You can set custom patterns to oekaki by storing them in the theme file. For example, by assembling the following tal file into .theme:

( theme )
fb70 fb70 fb70

( custom patterns )
7c10 1028 c701 0182 4444 4281 1028 4444
4844 4424 0810 1020 1122 4488 1122 4488

( diamond  brush )
0000 0010 0000 0000 0000 1038 1000 0000
0010 387c 3810 0000 1038 7cfe 7c38 1000
